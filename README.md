## 🔬 Implementation Details

### Question 1 — Manual MFCC Engine (no librosa)

Full pipeline implemented from scratch:

| Step | Method | Parameter |
|---|---|---|
| Pre-emphasis | `signal[n] - 0.97 × signal[n-1]` | α = 0.97 |
| Windowing | Hamming window | Frame = 25ms, Hop = 10ms |
| FFT | `numpy.fft.rfft` | N = 512 |
| Mel Filterbank | Triangular filters on Mel scale | 26 filters |
| Log Compression | `log(energy + 1e-8)` | — |
| DCT | Manual DCT-II | 13 coefficients |

**Spectral Leakage Results:**
| Window | SNR (dB) | Leakage Ratio |
|---|---|---|
| Rectangular | 16.325 | 0.338 |
| Hamming | 15.002 | 0.430 |
| Hanning | 14.398 | 0.435 |

**Phoneme Boundary RMSE:** `117.12 ms` between manual cepstrum boundaries and Wav2Vec2 forced alignment

---

### Question 2 — Disentangled Speaker Recognition

Based on: [Disentangled Representation Learning for Environment-agnostic Speaker Recognition](https://arxiv.org/abs/2406.14559)

| Model | Architecture | Loss Function |
|---|---|---|
| Baseline CNN | Conv1D × 2 → AvgPool → FC | Cross-Entropy only |
| Disentangled | Conv1D × 2 → AvgPool → Speaker head + Environment head (GRL) | CE + 0.1 × Environment CE |

**Key idea:** Gradient Reversal Layer (GRL) forces the encoder to suppress environment-specific information, making speaker embeddings environment-agnostic.

**Training config:**
- Epochs: 8
- Batch size: 16
- Optimizer: Adam (lr=1e-3)
- GRL lambda: Linear schedule 0 → 1

---

### Question 3 — Ethical Auditing & Privacy

**Bias Audit:** Documentation debt analysis across gender, age, and accent metadata fields. Measures percentage of missing/unknown labels as a proxy for representation bias.

**Privacy Module:** Signal-processing-based voice transformation:
| Profile | Pitch Shift | Formant Scale |
|---|---|---|
| Male → Female | +3 semitones | 1.15× |
| Female → Male | −3 semitones | 0.85× |
| Old → Young | +2 semitones | 1.10× |
| Young → Old | −2 semitones | 0.90× |

**Fairness Loss:**
```
L_total = L_CE(mean) + λ × Var(L_CE per demographic group)
```
Minimizes variance of cross-entropy loss across gender groups to equalize model performance.

## ⚠️ Troubleshooting

| Error | Fix |
|---|---|
| `DLL initialization failed` | Reinstall torch: `pip install torch==2.1.0 --index-url https://download.pytorch.org/whl/cpu` |
| `NumPy 2.x conflict` | Downgrade: `pip install "numpy<2"` |
| `Dataset scripts not supported` | Clear cache: `rmdir /s /q C:\Users\USERNAME\.cache\huggingface` |
| `torchcodec` error | Downgrade datasets: `pip install "datasets==2.19.0"` |
| `boundaries.npy not found` | Run `voiced_unvoiced.py` before `phonetic_mapping.py` |
| `baseline.pt not found` | Run `train.py` before `eval.py` |
| Symlinks warning on Windows | Harmless — safely ignore |
| HF_TOKEN warning | Harmless — safely ignore |
